# ResumeIQ  
### Intelligent Resume Screening & Candidate Ranking System  

ResumeIQ is a modern AI-powered web application designed to automate resume screening and candidate ranking using Natural Language Processing (NLP) and Machine Learning.  
It helps recruiters and hiring teams quickly identify the most relevant candidates based on a given job description.
